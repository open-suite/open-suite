## Change

Deploy pulls `ghcr.io/open-suite/*` images at pinned tags (CI already builds the frontend; add backend-overlay and auth-gate images to CI — auth-gate build context is `overlays/auth-gate/`). Remove on-box docker builds, `k3s ctr` imports, `:local` tags, `imagePullPolicy: Never`. Set the image via the env yaml/values so helmfile owns it, not `kubectl set image`.

- `kubectl get deploy -A -o yaml | grep -c 'imagePullPolicy: Never'` is 0.
- Pods survive `helmfile apply` with the right image.
