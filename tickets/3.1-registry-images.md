## Change

Remove on-box docker builds, `k3s ctr` imports, `:local` tags, `imagePullPolicy: Never`. Set the image via the env yaml/values so helmfile owns it, not `kubectl set image`.

### BLOCKER (2026-07-02): GHCR package visibility

CI publishes `ghcr.io/open-suite/auth-gate` but the package is created private,
so the box's anonymous `docker pull` returns 401. Before the auth-gate deploy
works off GHCR, the org owner must either:
- make the package public (GHCR → package → Package settings → Change
  visibility → Public), matching the OSS-distribution intent; or
- add an imagePullSecret (GHCR PAT with read:packages) to mb-bureaublad and
  reference it on the deployment.
Same step will apply to portal/meet images when they migrate. Not doable from
this session (token lacks `packages` scope).
