## Change (decided)

Remove the `OPEN_SUITE_DEMO_ADMIN_PASSWORD:-$MASTER_PASSWORD` fallback. Create a dedicated `demoadmin` Keycloak account with a generated password persisted to `/etc/mijnbureau/demo-admin-password` (mode 600). The login-page panel renders admin creds only when the env var is explicitly set; unset → panel shows only johndoe.
