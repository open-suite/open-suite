## Change (decided)

Validate the Bearer token — add PyJWT + cryptography to the image, verify signature against the realm JWKS (`https://id.<domain>/realms/mijnbureau/protocol/openid-connect/certs`, cached ~10 min) plus `iss` and `exp`; skip `aud` (mixed clients). Invalid/absent token → normal gate flow. Do not drop the passthrough (service-to-service API calls behind gated ingresses need it).

Also in the same file: stop reflecting arbitrary `Origin` in `send_preflight` (lines 213-222) — allow only `https://*.<domain>` origins.
